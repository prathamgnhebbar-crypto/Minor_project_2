# 💰 SpendDNA – Personal Finance Analytics

## 📌 Overview
SpendDNA is a Python-based Personal Finance Analytics project developed as part of my internship at **Unlox Academy**. The project analyzes six months of bank transaction data to identify spending patterns, categorize expenses, detect anomalies, and generate meaningful financial insights.

---

## 🎯 Objectives
- Clean and preprocess transaction data
- Extract vendors from transaction descriptions
- Categorize expenses into meaningful groups
- Analyze monthly spending trends
- Perform time-of-day spending analysis
- Detect unusual transactions using Z-Score
- Generate a comprehensive SpendDNA Report

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Jupyter Notebook
- Microsoft Excel

---

## 📂 Dataset
The dataset contains six months of personal bank transactions with the following fields:
- Date
- Time
- Description
- Transaction Type
- Amount
- Balance
- Payment Mode
- Reference Number
